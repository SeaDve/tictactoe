## Description 

A tic-tac-toe game written in Python and GTK3

## To test

To test, just download the `main.py` file from src and run it with python in the terminal.

`python main.py`

## Build from source

Clone this repo and run with GNOME Builder.

### Requirements

- Python 3 `python`
- PyGObject `python-gobject`
- GTK3 `gtk3`
- Meson `meson`
- Ninja `ninja`
